# Cycle Checklist

1. Freeze candidate opportunity list and record snapshot ID.
2. Randomize display order and remove implementation hints.
3. Distribute rubric + scoring sheet.
4. Collect scores and disagreement logs.
5. Run `npm run review:summary -- <path-to-scores.csv>`.
6. Run `npm run review:dashboard-data`.
7. Publish summary, dissent themes, and remediation backlog.
8. Start next cycle only after remediation owner assignment.
